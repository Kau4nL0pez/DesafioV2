Sistema de Operações Bancárias Tecnologias: Python, Programação Orientada a Objetos (POO).
Desenvolvimento de um sistema Back-End básico para gerenciar operações bancárias (saque, depósito, extrato). 
A arquitetura foi construída utilizando Programação Orientada a Objetos (POO) para definir classes de Cliente e Conta. 
Implementação de regras de negócio para validação de limite diário de saques e verificação de saldo, demonstrando aplicação de lógica condicional e encapsulamento.
